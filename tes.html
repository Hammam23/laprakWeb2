1. Struktur Dasar Halaman ▾

Jawaban: Buat satu file HTML, <h1> = LAPORAN PRAKTIKUM, <h2> berisi Nama | NIM, dan beberapa <h3> untuk tiap subbagian yang masing-masing memuat jawaban soal.

2. Interaktivitas Menggunakan JavaScript ▾

Jawaban: Ketika user mengklik judul subbagian (h3), tampilkan confirm() yang menanyakan apakah konten ingin disembunyikan. Jika "Ya" maka sembunyikan teks di bawahnya (mengubah atribut data-hidden), jika "Tidak" biarkan tetap tampil. Juga disediakan tombol untuk mengganti mode tampilan antara Rapi dan Berantakan.

3. Desain Tampilan Menggunakan CSS ▾

Jawaban: Gunakan Flexbox/Grid untuk mengatur tata letak. Atur warna latar, margin, padding, ukuran teks, dan transisi sederhana. Pilih font sans-serif yang mudah dibaca dan hindari kombinasi warna menyilaukan.

4. Responsivitas dengan RWD ▾

Jawaban: Tambahkan media queries:

  • Desktop (≥1024px): tampilkan dua kolom.
  • Tablet (600–1023px): satu kolom, ukuran teks menengah.
  • Smartphone (≤599px): satu kolom, ukuran teks dan tombol diperbesar untuk mudah disentuh.

Ringkasan / Lampiran ▾

Contoh ringkasan: Halaman ini adalah implementasi langsung dari kriteria praktikum. Klik judul subbagian untuk coba fitur konfirmasi & sembunyikan. Gunakan tombol "Mode Berantakan" untuk melihat perbedaan gaya.
