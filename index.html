1. Struktur Dasar Halaman ▾

Buat satu file HTML dengan judul utama <h1> “LAPORAN PRAKTIKUM”, subjudul berisi nama dan NIM, serta beberapa subbagian <h3> yang masing-masing berisi jawaban soal.

2. Interaktivitas Menggunakan JavaScript ▾

Gunakan confirm() saat klik subjudul. Jika “Ya”, sembunyikan teks; jika “Tidak”, biarkan tetap tampil. Tambahkan juga tombol untuk mengganti mode tampilan Rapi dan Berantakan.

3. Desain Tampilan Menggunakan CSS ▾

Gunakan Flexbox atau Grid untuk tata letak rapi, atur warna lembut dan font sans-serif agar nyaman dibaca. Tambahkan efek transisi ringan untuk memperhalus perubahan.

4. Responsivitas dengan RWD ▾

  • Desktop (≥1024px): dua kolom.
  • Tablet (600–1023px): satu kolom ukuran teks sedang.
  • HP (≤599px): satu kolom, tombol lebih besar untuk sentuhan.

Ringkasan / Catatan ▾

Halaman ini memenuhi seluruh kriteria praktikum: struktur HTML, interaktivitas JS, mode tampilan, dan desain responsif menggunakan CSS Grid dan media query.
